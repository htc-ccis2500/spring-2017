Visual Studio Mobile Tools

An introduction to cross-platform mobile development in Visual Studio.

Please take a few moments to learn about the options that we have for cross-platform mobile development in Visual Studio 2015.

As you watch the video, make notes on things that are not familiar to you so that you can ask questions and locate additional information to learn more about how various mobile apps are developed and deployed.
